# spring-security-demo
Ajax기반 Login

JWT를 이용한 회원 인증

동적으로 url권한 추가 및 삭제 가능

## 사용 기술
* Spring boot
* Spring security
* myBatis
* OAuth0 - jwt
* mySql
* Gradle
* Lombok
* Intellij Http Test

## 실행방법
1. db.sql문 mySql에 실행
2. 설정정보는 따로 손대지 않아도 작동

os - Ubuntu 16.04

java - 1.8.0_61

gradle - 4.0.1

spring boot - 1.5.10.RELEASE

mySql - 5.7.22

myBatis - 3.4.6
